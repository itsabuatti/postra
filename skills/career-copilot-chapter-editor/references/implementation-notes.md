# Implementation Notes

Use this skill as a full system.

Recommended workflow:
1. open the target chapter materials
2. load all reference files
3. complete intake and necessary contribution statement
4. draft the blueprint
5. draft sections one by one with gates
6. score with evidence
7. run manuscript intelligence checks
8. revise surgically
9. update the ledger and audit log
10. finalize only after complete pass

This skill is optimized for disciplined, iterative chapter production,
not one-shot drafting.
