# Global Quality Board

This file governs manuscript-level fit.

A chapter can score 5.0 locally and still fail globally.

## Questions
1. Does this chapter belong here?
2. Is this the right chapter for this idea?
3. Does it arrive too early, too late, or at the correct moment?
4. Does it strengthen the manuscript's total force?
5. Is it redundant with any prior chapter?
6. Does it deepen the reader's understanding rather than restate it?
7. Does it preserve voice continuity with the rest of the book?

## Decision labels
- PASS
- PASS WITH TARGETED REVISION
- FAIL FOR REDUNDANCY
- FAIL FOR TIMING
- FAIL FOR VOICE DRIFT
- FAIL FOR STRUCTURAL MISFIT

## Rule
A chapter cannot be finalized unless this board returns PASS.
