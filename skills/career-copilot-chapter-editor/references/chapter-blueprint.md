# Chapter Blueprint

Use this section architecture and minimums unless the user explicitly
instructs a justified structural variation.

## Section order and minimum lengths

I.   Opening Narrative | minimum 450 words
II.  Context Bridge | minimum 350 words
III. M.O.V.E. Framework Application | minimum 600 words per phase
IV.  Coaching Lab | minimum 550 words
V.   Career Co-Pilot Question Bank | minimum 350 words
VI.  Tactical Tool | minimum 450 words
VII. Chapter Closing | minimum 275 words
     APA 7 Reference List | complete and verified

## Science Snapshots — placement rule
Exactly 3 Science Snapshots are required per chapter. They are embedded
within the M.O.V.E. section, not placed as a standalone section. Each
snapshot follows the M.O.V.E. phase it supports. Typical placement:
one after MAP, one after OPERATE, one after VERIFY. The EVOLVE phase
does not require a Science Snapshot.

Each snapshot: minimum 175 words. Label with ✦ SCIENCE SNAPSHOT.
Include an ESSENCE line and a CO-PILOT ACTION (not "Parent Action")
line after the Essence.

## Inline canvas format
When the M.O.V.E. section includes the Career Canvas, display each
canvas section as a labeled, inline fillable form within the phase
text — not as a separate appendix. Canvas sections are titled:
M.O.V.E. CAREER CANVAS — SECTION [#]: [PHASE NAME]

## Tactical Tool structure
The Tactical Tool section uses named subsections, not a single
undifferentiated block. Typical subsections for the Career Canvas tool:
- How and When to Use Each Section
- A Note on the [key tool element, e.g., Story Field in EVOLVE]
- Co-Pilot Ground Rules for [Tool] Conversations

## Heading label rule
Section II must be labeled "Context Bridge" or "The Context Bridge."
The label "Ground Control Bridge" is an aviation violation under
IRON RULE 1 and must never appear.

## Blueprint rule
Structure is part of the pedagogy. Do not collapse sections casually.
