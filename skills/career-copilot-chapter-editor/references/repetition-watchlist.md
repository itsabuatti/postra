# Repetition Watchlist

Track overused language, structures, and argument patterns.

## Watch for
- repeated sentence openings
- repeated explanation of the same framework with no added depth
- repeated story types
- repeated emotional beats
- repeated terminology clusters
- repeated chapter endings that resolve in the same way

## Classification
A. Intentional reinforcement
B. Useful extension
C. Redundant restatement

If classified as C:
- remove
- condense
- deepen
- reframe in a materially new way
